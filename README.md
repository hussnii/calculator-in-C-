# calculator-in-C++
Calculator System in C++

A simple console-based calculator system developed using C++.
This project performs different mathematical operations through a menu-driven interface.

📌 Features

The calculator supports the following operations:

Arithmetic Operations
Addition
Subtraction
Multiplication
Division
Power Calculation
Calculates a number raised to another number
Sum of Digits
Finds the sum of all digits in a number
Sum of Natural Numbers
Calculates the sum of first N natural numbers
Factorial Calculation
Finds factorial of a given number
Even or Odd Checker
Checks whether a number is even or odd
🛠️ Technologies Used
C++
Functions
Loops
Conditional Statements
Switch Case
Menu Driven Programming

▶️ How to Run the Program
Step 1: Compile the Program

Using g++ compiler:

g++ main.cpp -o calculator
Step 2: Run the Program
./calculator

For Windows:

calculator.exe
